# 项目简介

## 核心功能
- 集成百度地图
- 首页文创产品推荐（协同算法，按用户购买行为推荐）
- 用户管理：增删改查用户信息
- 文创产品管理：订单处理、产品上架
- 文创活动管理
- 文化资讯发布
- 定制设计需求处理
- 用户端文创信息推荐、公告查看、文创产品购买与支付
- 参与文创活动：预约、收藏、活动地图查看
- 浏览文化资讯
- 社交分享模块：发布、回复、评论帖子
- 个人中心：管理账户，查看收藏、订单、预约信息

## 适用场景
- 文化创意产业
- 电商平台
- 社交媒体

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
